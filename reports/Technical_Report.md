# Technical Report

## Project Name
AI Fake News Detection

## Objective
Develop an AI system to detect fake news articles.

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- GitHub

## Dataset
- Fake.csv
- True.csv

## Project Workflow
1. User enters a news article.
2. Frontend sends the request to the backend.
3. Backend processes the request.
4. AI model predicts whether the news is fake or real.
5. The result is displayed to the user.

## Output
The system predicts whether the news is Fake or Real.
